# Composite

Use this as the relationship-as-an-entity reading.

## Primary reference

- `synastry.md` — Part 2 covers the full composite framework (reading the composite as a natal chart, key composite patterns, composite vs. synastry)

## Focus

- Relationship identity: how the bond presents (composite Ascendant) and what it is for (composite Sun)
- Relationship inner life: what the bond needs to feel safe (composite Moon)
- Shared pressure points: where the bond must work hardest (composite Saturn)
- The bond as a single system with its own paradox — name it, the same way a natal reading names the chart's central tension

## Data notes

- The `relationship_themes[]` array in the script output contains extracted composite themes — use as starting points, then deepen
- Read synastry first when both are available: synastry is how the two people interact; composite is what the relationship itself is. A difficult synastry with a strong composite can still build something lasting.

## Asteroid layer

- Juno: what binds the bond
- Vesta: what the bond protects
- Ceres: how the bond nourishes
- Pallas: how the bond solves problems
