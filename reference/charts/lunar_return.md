# Lunar Return Chart Reading Framework

A lunar return occurs when the transiting Moon returns to the exact degree and sign of the natal Moon — approximately every 27.3 days. The chart cast for that moment describes the themes, opportunities, and challenges of the **upcoming lunar month** (~4 weeks).

---

## What a Lunar Return Is

Unlike the natal chart (lifetime) or a transit chart (current sky), the lunar return is:
- **Temporary**: valid for ~4 weeks until the next lunar return
- **Personal**: tied to the individual's natal Moon position
- **Location-sensitive**: some astrologers deliberately travel to favorable locations before their lunar return to improve the return chart

---

## Reading Sequence

### Step 1 — Lunar Return Ascendant

The Ascendant of the return chart sets the tone for the entire month.

| Ascendant Sign | Monthly Tone |
|----------------|-------------|
| Aries | Active, assertive month; new initiatives begin |
| Taurus | Stable, sensory month; focus on comfort and resources |
| Gemini | Busy, communicative month; multiple irons in the fire |
| Cancer | Emotionally deep month; home and family prominent |
| Leo | Creative, expressive month; visibility and recognition |
| Virgo | Productive, detail-oriented month; health and routines |
| Libra | Relational month; partnerships and decisions |
| Scorpio | Intense, transformative month; depth and power |
| Sagittarius | Expansive, adventurous month; learning and travel |
| Capricorn | Disciplined, ambitious month; career and goals |
| Aquarius | Unconventional month; groups and social causes |
| Pisces | Intuitive, reflective month; creativity and spirituality |

Note which natal house this Ascendant activates (i.e., where this sign falls in the natal chart) for a double reading.

### Step 2 — Moon in Return Chart House

The house where the Moon falls in the return chart shows the **primary focus** of the month.

| Moon in House | Monthly Focus |
|---------------|--------------|
| 1st | Personal needs, self-expression, new identity work |
| 2nd | Finances, self-worth, material needs |
| 3rd | Communication, writing, local connections, siblings |
| 4th | Home, family, private life, emotional processing |
| 5th | Romance, creativity, children, pleasure |
| 6th | Health, daily work, service, routines |
| 7th | Relationships, partnerships, dealing with others |
| 8th | Deep emotions, shared resources, transformation |
| 9th | Learning, travel, philosophy, expansion |
| 10th | Career, public life, reputation, goals |
| 11th | Friendships, groups, hopes and community |
| 12th | Retreat, reflection, spiritual work, rest |

### Step 3 — Return Chart Sun Placement

The house and sign of the Sun in the return chart indicates where **vitality and focus** are directed.

Cross-reference with:
- The natal house where this sign falls
- Any aspects the return Sun makes to natal or return planets

### Step 4 — Major Aspects in Return Chart

Apply the aspect interpretation from `aspects.md`. Pay special attention to:

1. **Moon aspects** (return Moon to return planets): emotional dynamics of the month
2. **Return Sun aspects**: vitality and attention themes
3. **Return Venus and Mars**: relationship and desire trends
4. **Any planet conjunct the Angles** (Ascendant, IC, Descendant, Midheaven): dominates the month

**Return planet conjunct natal planet**: An especially personal activation — this area is highlighted this month.

### Step 5 — Monthly Themes

The `return_time` from the ephemeris output gives the exact moment the Moon returned. Use this to frame the reading as a new monthly chapter.

Walk through 2–4 key themes in order of impact:
1. The most powerful aspect (tightest orb or most angular)
2. The Moon's house message
3. The Sun's direction
4. Any standout pattern (stellium, grand trine, etc.)

### Step 6 — Connections to Natal Chart

Overlay the return chart planets onto the natal houses to deepen the reading:

- A return planet in the natal 10th house → career highlighted this month
- Return Venus in the natal 7th house → relationship themes heightened
- Return Saturn on the natal Ascendant → a month of responsibility and new structure
- Return Jupiter conjunct natal Moon → emotional expansion, good fortune in personal life


## Delivering the Reading

Structure:
1. **Return moment**: "Your lunar return this month occurs on [date/time] at [location]."
2. **Monthly tone** (return Ascendant): one sentence on the overall atmosphere
3. **Core focus** (Moon's house): what emotional and practical area is lit up
4. **Key themes** (2–4 major patterns): what the month wants to teach or offer
5. **Opportunities**: where to lean in
6. **Cautions**: where to be careful
7. **Closing**: a practical suggestion for the month ahead
