# Asteroids

Use these when the chart includes meaningful asteroids. Gather them into one dedicated chapter of the reading, not scattered mentions.

## Core meanings

- **Ceres**: nurture, food, care, attachment, loss and recovery
- **Pallas**: strategy, pattern recognition, craft, intelligent defense
- **Juno**: commitment, reciprocity, marriage, contracts, what partnership asks to be taken seriously
- **Vesta**: devotion, focus, sacred work, boundaries, what must remain undivided

## Reading rules

- Read asteroid sign, house, and tight aspects (within ~3°)
- Give them secondary weight after Sun/Moon/Ascendant
- Prioritize: angular houses; the 5th/7th/8th/11th/12th; tight aspects to personal planets, angles, Nodes, or Chiron
- Translate into lived themes, not catalog keywords
- If an asteroid has no meaningful placement and no tight aspect, mention briefly or omit

## Tight-aspect reading patterns

When a tight aspect exists, read the asteroid through that contact:

| Contact | Theme |
|---------|-------|
| Juno + Venus/Mars/Saturn | Commitment pattern — what binds, and at what price |
| Ceres + Moon/Venus/Saturn | Attachment / nourishment pattern — how care was given, withheld, or lost |
| Vesta + Sun/Mars/ASC | Devotion / self-containment pattern — the flame that must be protected |
| Pallas + Mercury/Saturn/MC | Strategy / pattern-recognition — how intelligence organizes survival |
