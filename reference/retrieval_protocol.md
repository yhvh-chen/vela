# Retrieval Protocol

Retrieval is optional depth, not a dependency.

## Modes

- **Chart + retrieval**: use retrieved context to sharpen interpretation
- **Chart-only**: use chart facts plus local references only

## Limits

- Start from chart anchors first
- Keep retrieval focused
- Prefer small, category-specific queries
- Never let retrieval replace actual chart evidence

## Use

- Use retrieval when it clearly improves nuance
- Skip it cleanly when unavailable
- Keep the same disclosure order in both modes
